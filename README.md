# 🧮 Neumorphic Calculator with History

A sleek, modern, and interactive calculator built using **HTML5**, **CSS3 (Neumorphic Design)**, and **Vanilla JavaScript**. This project features a clean dark-themed UI with a functional sidebar to track your recent calculation history.

## ✨ Features

* **Neumorphic UI:** Soft shadows and highlights for a realistic, 3D-pressed effect.
* **Calculation History:** Automatically saves your last calculations in a dedicated side panel.
* **Interactive History:** Click on any past result to bring it back to the main display.
* **Fully Responsive:** Works perfectly on desktops and adjusts for mobile screens.
* **Smart Functions:** Includes standard operations, percentage, delete (DEL), and All Clear (AC).

## 🚀 Technologies Used

* **HTML5:** Structure and semantic layout.
* **CSS3:** Custom styling, Flexbox, and Neumorphic box-shadows.
* **JavaScript:** Logic for calculations, DOM manipulation, and history management.

    Simply open `index.html` in your favorite web browser.

## 📜 How it Works

The calculator uses the `eval()` function in JavaScript to process mathematical expressions. The history is managed dynamically by creating new DOM elements (`div`) every time the `=` button is pressed, ensuring a seamless user experience.
